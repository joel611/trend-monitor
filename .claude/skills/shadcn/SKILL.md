### when to use me
Use this when you going to create shadcn component

### Steps
- verify shadcn is proper initialized
- run shadcn cli to add component

### Useful command

#### init shadcn
```
bunx --bun shadcn@latest init
```

#### add component
```
bunx --bun shadcn@latest add <component>  # cli to add new  component
bunx --bun shadcn@latest add card  # add card component
bunx --bun shadcn@latest add button  # add button component
```
